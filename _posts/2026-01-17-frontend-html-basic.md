HTML(HyperText Markup Language)은 웹 페이지의 구조를 의미하는 기본적인 언어다. (디자인 요소는 CSS에서 담당)

### HTML의 역할
- **구조:** 텍스트, 이미지, 버튼 등 페이지에 들어갈 요소를 정의
- **계층 구조:** 태그 안에 태그를 넣는 부모/자식 관계를 통해 형성

### 시맨틱 태그
태그 이름만으로 그 요소가 어떤 역할을 하는지 알려준다 - 검색엔진 최적화, 유지보수 효율성


#### 예시

| 태그 | 역할 |
| :--- | :--- |
| **`<header>`** | 페이지나 섹션의 상단 부분(로고, 메뉴 등) |
| **`<nav>`** | 네비게이션 링크 |
| **`<main>`** | 문서의 핵심 콘텐츠(페이지당 한 번만 사용) |
| **`<section>`** | 문서의 독립성에 따라 나눌 때 |
| **`<article>`** | 뉴스, 기사 |
| **`<aside>`** | 사이드바 등 부수적인 정보 |
| **`<footer>`** | 페이지 하단의 저작권, 주소, 연락처 등 |

---

`div`, `span` 사용보다는 적재적소에 시맨틱 태그를 활용하기 - 협업 효율성 높임
